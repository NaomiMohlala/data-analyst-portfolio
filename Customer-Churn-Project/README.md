Customer Churn Analysis Dashboard

📊Overview
This project analyzes customer churn data to identify patterns and key factors influencing customer retention.

🔍Key insights
month-to-month contracts have the highest churn rate.
High monthly charges increase churn risk.
Long-term contracts improve retention.

💡Recommendations
Offer incentives for long-term contracts.
Improve onboarding for customers.
Adjust pricing strategies.

🛠Tools used
Micrsoft Excel
Pivot Tables
Data Visualization
